Run a LLM model with Ollama
1. Install Ollama: https://ollama.com/download
2. Pull a LLM model
   # ollama pull llama3.2:1b
3. Run pulled LLM model
   # ollama run llama3.2:1b

Interract with LLM model with API
4. Install Postman: https://www.postman.com/downloads/
5. Enable Ollama service
   # ollama serve
6. Interact with Ollama using API
   - Download ollama-rest-api-collection.json
   - Import to Postman
   - Create Postman environment variables
   - Lets play!
