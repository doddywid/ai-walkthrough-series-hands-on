Lab preparation
1. Install Ollama: https://ollama.com/download
2. Pull a LLM model
   # ollama pull llama3.2:1b
3. Install Postman: https://www.postman.com/downloads/

Run LLM model
4. Run pulled LLM model
   # ollama run llama3.2:1b

Interract with LLM model with API
5. Enable Ollama service
   # ollama serve
6. Interact with Ollama using API
   - Download ollama_rest_api_collection.json
   - Download ollama_env.json
   - Import collection & environment files to Postman
   - Lets play!
